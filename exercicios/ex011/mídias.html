Imagens dinâmicas

Tente abrir esse site em dispositivos diferentes, ou apenas aumente e diminua o tamanho do navegador

Para isso funcionar,siga os passos:

  1. basta adicionar a tag PICTURE
  2. depois usar a tag IMG
  3. selecionar o arquivo que deseja
  4. logo após, adicione a tag SOURCE:MEDIA:TYPE
  5. escreve max no lugar de min
  6. selecione os pixels e dps escreva o nome do arquivo

Reproduzindo aúdios

Vamos aprender a reproduzir aúdios em HTML5

Para acrescentar um aúdio no site, siga esses passos:

  1. digite a tag AUDIO
  2. aperte ctrl + espaco para selecionar o arquivo de aúdio
  3. escreva os comando AUTOPLAY para o aúdio tocar sozinho (nao está funcionando) e o comando CONTROLS.

    4. Para ter mais compatibilidade:

    1. coloque a tag AUDIO
    2. apague o SRC
    3. escreva no lugar, PRELOAD""
    4. dentro das aspas, escreva METADATA
    5. depois escreva os mesmos comandos do método anterior
    6. após isso, insira a tag SOURCE:SRC
    7. aperte ctrl + espaco para selecionar o arquivo de aúdio.
    8. ao lado do TYPE, escreva AUDIO/compactacao (mpeg,ogg,wav)
