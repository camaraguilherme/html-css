Trabalhando com listas

Listas Ordenadas

Para criar uma lista ordenada, basta utilizar a tag OL e depois utiliza-se a tag LI.

Caso queira modificar as numeracoes, basta colocar type dentro da tag e escolher o tipo, sendo eles:
A: Letras maiúsculas
a: "" minúsculas
I: Algarismo Romano grande
i: "" pequeno
1: Números comecando do 1
(para comecar a lista a partir de qualquer outro núm, basta inserir apenas o núm que deseja).

EXEMPLO:

  1. acordar
  2. tomar café
  3. estudar
  4. ir para a faculdade
  5. almocar

Listas nao Ordenadas

Para criar uma lista nao ordenada, basta utilizar a tag UL

Caso queira modificar os símbolos, basta colocar type dentro da tag e escolher o tipo, sendo eles:
-disc: disco preenchido
-circle: disco nao preenchido
-square: quadrado

EXEMPLOS:

  • batata frita
  • katchup
  • queijo

Juntando listas

  1. god of war
    • Ragnarok
    • GOW 1
    • GOW 2
